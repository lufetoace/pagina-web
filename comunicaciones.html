COMUNICACIONES

ESTACIÓN METEREOLÓGICA

MODULO DE WIFI

Se utilizara un modulo WIFI (ESP 8266), que estará conectado a un arduino, este modulo es programado mediante códigos AT (son instrucciones codificadas que conforman un lenguaje de comunicación entre el hombre y un terminal modem), la primera parte de dicha programación se basa en la asignación de una red WIFI al modulo que se realiza sin el microcontrolador del arduino porque esto se cargara directamente al ESP8266. después se adiciona otra vez el microcontrolador al arduino para cargar el código de envió de datos.

UBICACION EN LA INTERFAZ MECÁNICA

El modulo WIFI se conectara con las raspberry mediante un cable USB, esta cable estará forado por una manguera que lo protegerá de las filtraciones de agua que pueda tener, además este modulo se encuentra ubicando en una boya y la raspberry en un modulo sumergible a 40 cm de la superficie del agua. La transmisión y recepción de datos se realizara estando conectado a una red local de internet.

Edup usb

Debido a las limitaciones de memoria (96 [kB]) del módulo ESP-8266 no se pudo implementar para transmitir de todos los datos del proyecto temperatura, presión, pH e imágenes de la cámara. Por esta razón se cambió de módulo wifi y se utilizó uno llamado “Edup usb” de referencia 802.11N, la ventaja de este módulo es que no necesita configuración previa y tiene gran capacidad para enviar todos los datos del proyecto.
